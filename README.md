# API Debug OpenEnv

A real-world OpenEnv environment that simulates production API debugging tasks.

## Features
- OpenEnv-compliant `reset`, `step`, `state`
- 3 tasks: easy, medium, hard
- Partial reward signals
- FastAPI server
- Docker-based Hugging Face Space
